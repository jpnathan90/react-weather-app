Weather APP Task

## Installation

After you clone the repo, install all the necessary npm dependencies:

```
npm install
```

or

```
yarn
```

## Running the App

To run the App

```
npm run start
```

or

```
yarn start
```

Then, open the page [http://localhost:3000](http://localhost:3000) in your web browser.
